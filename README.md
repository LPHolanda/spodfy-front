# Spodfy

## Objetivo
> Criar uma aplicação que consome a API pública do Spotify®, que permite que o usuário se autentique com seu perfil e selecione seus poscasts favoritos, para que possa ser notificado toda vez que um episódio novo do podcast é lançado.

## Execução CLI
A aplicação foi desenvolvida em Angular, e para subir o servidor, na pasta /spodfy-webapp, execute o seguinte comando
```bash
$  ng serve --o
```
